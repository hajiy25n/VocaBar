# Changelog

VocaBar의 모든 주요 변경사항을 기록한다.

형식은 [Keep a Changelog](https://keepachangelog.com/ko/1.0.0/)를 따르고,
버전 번호는 [Semantic Versioning](https://semver.org/lang/ko/)을 기반으로 한다.

## [v1.4] - 2026-03-28

### Added
- **폴더/카테고리 분류**: 단어를 폴더별로 분류, 설정에서 활성 폴더 선택, 단어 추가 시 다중 폴더 선택 가능
- **퀴즈 오늘의 단어 전용**: 퀴즈가 오늘의 단어에서만 출제됨 (10개 미만이면 반복 출제로 10문제 채움)
- **퀴즈 자동 넘김**: 정답 시 1초, 오답 시 2초 후 자동으로 다음 문제 이동
- **퀴즈 진행도 색상 표시**: 오늘의 단어 목록에서 각 단어별 X/3 표시 (0 빨강 → 1 주황 → 2 노랑 → 3 초록)
- **일일 풀 안정화**: 목표 단어 수를 변경해도 기존 학습 단어는 유지, 풀에서 추가/제거만 됨
- **전환 간격 프리셋 버튼**: 5/10/15/30/60초 원클릭 설정 (기존 슬라이더도 유지)
- **100% 달성 축하**: 오늘의 목표 100% 달성 시 축하 메시지 표시
- **TOEFL 폴더 자동 배정**: 내장 샘플 단어가 "TOEFL" 폴더에 자동 배정
- VERSION_GUIDE.md (Git/버전 관리 가이드)
- CHANGELOG.md (이 파일)

### Changed
- **학습완료 단어 메뉴바 제외**: isLearned=true인 단어는 메뉴바 순환에서 자동 제외 (미학습 0개일 때만 전체 순환)
- **하이라이트 박스 고정 높이**: 예문 2줄 고정(32pt)으로 레이아웃 안정화
- **이모지 기기별 랜덤**: 기기마다 고유 시드 → 사용자마다 다른 이모지 순서, 3시간마다 변경
- README.md 간결하게 정리 (안정적인 개요만 유지)
